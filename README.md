# Análisis de Datos de Espectroscopía Raman Biomédica

## Descripción

Este proyecto presenta un pipeline de análisis de datos para espectroscopía Raman aplicada a muestras de piel utilizando MATLAB y SQL.

El flujo de trabajo incluye:

- Carga de datos espectroscópicos
- Corrección de línea base
- Reducción de ruido
- Suavizado de señales
- Extracción de regiones espectrales
- Ingeniería de características
- Creación de datasets
- Análisis mediante SQL
- Visualización de datos

--------------------------------

## Tecnologías Utilizadas

- MATLAB
- SQL (SQLite)
- CSV / Excel
- Procesamiento de señales
- Análisis de datos

-------------------------------

## Flujo de Procesamiento

1. Carga de datos
2. Preprocesamiento espectral
3. Corrección de línea base
4. Suavizado de señales
5. Extracción de picos Raman
6. Extracción de características
7. Generación de dataset
8. Consultas SQL
9. Visualización de datos

------------------------------

## Estructura del Repositorio

- `/scripts` → Scripts MATLAB del pipeline
- `/sql` → Consultas SQL para análisis
- `/exports` → Datasets exportados
- `/figures` → Resultados visuales

--------------------------------

## Aviso de Confidencialidad

Los datasets experimentales originales no se incluyen debido a restricciones de confidencialidad de investigación.

Este repositorio se enfoca únicamente en la metodología de procesamiento y análisis de datos.
